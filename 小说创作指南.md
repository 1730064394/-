
# 《都市：开局捡了个吐槽系统》创作指南

## 一、核心创作原则（番茄小说风格）

### 1.1 节奏要快
- 每3章一个小高潮
- 每10章一个大高潮
- 开篇必须抓住读者（前3章必须激活系统、获得第一个能力、解决第一个麻烦）

### 1.2 爽点密集
- 打脸情节要及时
- 升级速度要合理但不能太慢
- 主角要不断获得新能力、新装备、新身份

### 1.3 人物鲜活
- 主角性格要鲜明（乐观开朗、重情重义）
- 配角要有记忆点（逗比系统、高冷女主角、搞笑朋友等）
- 反派要足够坏，打脸要足够爽

### 1.4 代入感强
- 主角从底层起步，读者容易代入
- 情节贴近生活，容易引起共鸣
- 情感线要真实，不突兀

## 三、系统功能详解

### 3.1 吐槽任务
- 日常吐槽：吐槽身边的人和事，获得少量积分
- 特殊吐槽：针对特定人物或事件的吐槽，获得大量积分和特殊奖励
- 极限吐槽：在危险情况下吐槽，奖励翻倍，但风险也高

### 3.2 选择系统
- 关键时刻给出2-4个选项
- 不同选项对应不同奖励和剧情走向
- 有最优选项和普通选项，考验读者判断力

### 3.3 系统商城
- 异能类：各种异能和异能升级
- 技能类：格斗术、医术、黑客技术等
- 物品类：各种道具、装备
- 特殊类：血统、传承等稀有物品

### 3.4 属性面板
```
宿主：林凡
力量：15（F级力量强化+10）
速度：10
体质：12
精神：12
异能：力量强化（F级）
积分：2500
技能：基础格斗术（精通）、治愈术（入门）
```

## 五、番茄小说写作技巧

### 5.1 标题和简介
- **标题：** 要包含核心卖点，如《都市：开局捡了个吐槽系统》
- **简介：** 简明扼要，突出爽点和悬念，200字左右

**简介示例：**
> 林凡，一个在社会底层挣扎的孤儿，意外激活了神级吐槽系统！
> 只要吐槽就能获得积分和奖励！
> 富二代嚣张？吐槽他！
> 反派装逼？吐槽他！
> 从此，林凡在都市中开启了一段传奇人生！
> 且看他如何从底层崛起，守护身边的人，成为都市最强者！

### 5.2 章节开头
- 每章开头要有钩子，吸引读者继续看下去
- 可以用对话、场景描写、悬念等方式开头

**示例：**
> 「林凡，你给我等着！这件事没完！」赵坤撂下一句狠话，狼狈地钻进车里，扬长而去。
> 林凡看着远去的法拉利，嘴角露出一抹笑容：「想找我麻烦？我等着你！」

### 5.3 章节结尾
- 每章结尾要有悬念，让读者想看下一章
- 可以用反转、危机、新任务等方式结尾

**示例：**
> 就在林凡准备离开时，手机突然响了。
> 是孤儿院打来的！
> 「林凡哥哥，不好了！院长妈妈她……」
> 电话那头传来弟弟妹妹带着哭腔的声音。
> 林凡脸色一变，拔腿就跑！

### 5.4 对话技巧
- 对话要符合人物性格
- 对话要推动剧情发展
- 对话要有趣味性（尤其是系统的吐槽）

### 5.5 节奏把控
- 爽文节奏：铺垫→冲突→爆发→升级→新的铺垫
- 每章2000-3000字
- 每10章左右一个小卷结束

## 七、番茄小说平台注意事项

1. **字数要求：** 单章2000-3000字，总字数100万字以上
2. **更新频率：** 最好每天更新2-3章，稳定更新很重要
3. **互动：** 多和读者互动，回复评论
4. **数据：** 关注收藏、推荐票、评论等数据，根据读者反馈调整剧情
5. **上架：** 一般30万字左右可以申请上架
