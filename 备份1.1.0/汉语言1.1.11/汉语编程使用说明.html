汉语编程使用说明

1. 简介

汉语编程是一种使用中文关键字的编程语言，语法类似于Python，但使用中文关键字和语法结构。本使用说明将详细介绍汉语编程的语法、特性和使用方法。

2. 基本数据类型

2.1 整数

# 定义整数变量
定义 年龄 = 18
打印(年龄)  # 输出: 18

2.2 浮点数

# 定义浮点数变量
定义 身高 = 1.75
打印(身高)  # 输出: 1.75

2.3 字符串

# 定义字符串变量
定义 姓名 = "张三"
打印(姓名)  # 输出: 张三

# 字符串拼接
定义 问候语 = "你好，" + 姓名
打印(问候语)  # 输出: 你好，张三

2.4 布尔值

# 定义布尔值变量
定义 是学生 = 真
定义 是老师 = 假
打印(是学生)  # 输出: 真
打印(是老师)  # 输出: 假

2.5 空值

# 定义空值变量
定义 未定义变量 = 空
打印(未定义变量)  # 输出: 空

2.6 列表

# 定义列表
定义 数字列表 = [1, 2, 3, 4, 5]
打印(数字列表)  # 输出: [1，2，3，4，5]

# 访问列表元素
打印(数字列表[0])  # 输出: 1

# 修改列表元素
数字列表[0] = 10
打印(数字列表)  # 输出: [10，2，3，4，5]

2.7 字典

# 定义字典
定义 学生信息 = {"姓名": "张三", "年龄": 18, "性别": "男"}
打印(学生信息)  # 输出: {姓名: 张三，年龄: 18，性别: 男}

# 访问字典元素
打印(学生信息["姓名"])  # 输出: 张三

# 修改字典元素
学生信息["年龄"] = 19
打印(学生信息)  # 输出: {姓名: 张三，年龄: 19，性别: 男}

2.8 集合

# 定义集合
定义 数字集合 = {1, 2, 3, 4, 5}
打印(数字集合)  # 输出: {1，2，3，4，5}
